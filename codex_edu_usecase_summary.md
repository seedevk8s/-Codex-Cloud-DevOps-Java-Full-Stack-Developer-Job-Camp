
# OpenAI Codex 활용 요약

## 도입 목적
- 강의 콘텐츠 자동화
- 실습 과제 평가 자동화
- 실시간 코드 리뷰 및 디버깅 보조

## 활용 영역
### 프론트엔드
- HTML/CSS/JS 예제 자동 생성
- 실시간 DOM 조작 예제 시연

### 백엔드
- Spring Boot 구조 자동 제안
- 서비스/컨트롤러/Repository 보일러플레이트 생성

### DB
- SQL 쿼리 자동 생성 및 검토
- 데이터 스키마 설계 도우미

### 팀 프로젝트
- Git PR 리뷰 자동화
- 기능 구현 시 실시간 코드 보조

## 추천 프롬프트 예시
- "React로 기본적인 Todo 앱 만들어줘"
- "아래 코드에서 오류를 찾아줘"
- "이 SQL 쿼리를 최적화해줘"

## 결론
Codex를 도입하면 강사 입장에서 반복 업무를 줄이고, 학습자에게는 실시간 피드백과 코드 품질 향상의 기회를 제공할 수 있습니다.
