
# Lab: HTML & CSS 기본 실습

## 1. 실습 목표
- HTML 문서 구조 이해
- 시맨틱 태그 사용
- CSS로 스타일링 적용

## 2. 실습 요구사항
- 웹페이지에 header, nav, main, footer 영역을 배치
- Flexbox를 이용한 2단 레이아웃 구현

## 3. starter.html
```html
<!DOCTYPE html>
<html>
<head>
  <title>레이아웃 실습</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <!-- 여기에 코드 작성 -->
</body>
</html>
```

## 4. style.css 예시
```css
body { font-family: sans-serif; }
.container { display: flex; }
```

## 5. 제출형식
- index.html, style.css
