   - MSA 설계 및 구현
   - 컨테이너와 쿠버네티스 운영
   - 클라우드 서비스 활용

2. **💻 풀스택 개발 능력**
   - 프론트엔드: React 기반 SPA 개발
   - 백엔드: Spring Boot 기반 API 개발
   - 데이터베이스: 설계부터 운영까지

3. **⚙️ DevOps 실무 역량**
   - CI/CD 파이프라인 구축
   - 인프라 자동화 (Terraform)
   - 모니터링 및 로깅

4. **☁️ AWS 클라우드 전문성**
   - EC2, S3, RDS 등 핵심 서비스
   - Lambda를 활용한 서버리스
   - CloudWatch 모니터링
